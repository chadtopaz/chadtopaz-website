Getting Started with TDA

A curated list of resources for people who want to learn topological data analysis but have never taken topology. Aimed at those with similar background (or lack thereof) as Chad himself.

There are too many resources on TDA out there. Here is the list of resources Chad gives to people who ask. This is aimed at people who have not taken algebra, topology, or algebraic topology — because Chad hasn't either. It focuses on computation of persistent homology for point cloud data.

Recommended resources
  • Connecting the Dots: Discovering the Shape of Data — A totally nontechnical introduction to TDA.
  • Topological data analysis: One applied mathematician's heartwarming story of struggle, triumph, and ultimately, more struggle — An expository article, slightly more technical.
  • Topological data analysis of collective motion — An expository article.
  • Topological data analysis of biological aggregation models — Uses persistent homology for exploratory data analysis.
  • Self-help homology tutorial for the simple(x)-minded — Tutorial notes showing how computation of homology is really just linear algebra.
  • A topological approach to selecting models of biological experiments — Uses topological signatures to decide which of two models fits data better.
  • Analyzing collective motion with machine learning and topology — Applying machine learning to topological signatures to estimate parameters.
  • Introduction to the R Package TDA — A tutorial to work through.
